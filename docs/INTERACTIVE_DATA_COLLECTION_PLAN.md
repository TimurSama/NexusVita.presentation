# План интерактивных методов сбора данных

## Цель
Создать визуально привлекательные и интуитивно понятные интерактивные элементы для сбора информации о здоровье пользователя.

---

## Интерактивные элементы

### 1. Рост - Рулетка (Wheel Picker)
**Концепция:** Вертикальная рулетка с делениями, которую можно прокручивать
- Визуал: Классическая рулетка с делениями в см
- Интерактив: Прокрутка колесиком мыши, свайп на мобильных, клик и перетаскивание
- Анимация: Плавное вращение с инерцией
- Диапазон: 100-220 см
- Шаг: 1 см

### 2. Вес - Грузики (Weight Slider)
**Концепция:** Визуализация веса через грузики, которые можно добавлять/убирать
- Визуал: Грузики разного веса (1кг, 5кг, 10кг, 20кг)
- Интерактив: Перетаскивание грузиков на весы, клик для добавления/удаления
- Анимация: Грузики падают на весы, стрелка весов отклоняется
- Диапазон: 30-200 кг
- Шаг: 0.5 кг

### 3. Дата рождения - Календарь Майя (Mayan Calendar Wheel)
**Концепция:** Круглый календарь с концентрическими кольцами для года, месяца, дня
- Визуал: Круглый календарь с красивыми узорами
- Интерактив: Вращение колец для выбора даты
- Анимация: Плавное вращение колец, подсветка выбранной даты
- Формат: День/Месяц/Год

### 4. Артериальное давление - Манжета (Blood Pressure Cuff)
**Концепция:** Интерактивная манжета, которую нужно "накачать"
- Визуал: Медицинская манжета с манометром
- Интерактив: Нажатие/удержание для накачивания, выбор значений на манометре
- Анимация: Манжета надувается, стрелка манометра движется
- Формат: Систолическое/Диастолическое (120/80)

### 5. Пульс - Пульсометр (Heart Rate Monitor)
**Концепция:** Интерактивный пульсометр с визуализацией пульса
- Визуал: Пульсометр с графиком сердечного ритма
- Интерактив: Клик для "измерения", выбор значения на шкале
- Анимация: Пульсирующая линия, мигающий индикатор
- Диапазон: 40-200 уд/мин

### 6. Настроение - Эмодзи слайдер (Emoji Mood Slider)
**Концепция:** Горизонтальный слайдер с эмодзи настроения
- Визуал: Эмодзи от грустного до радостного
- Интерактив: Перетаскивание слайдера, клик на эмодзи
- Анимация: Эмодзи увеличиваются при наведении, плавное движение
- Шкала: 1-10

### 7. Уровень стресса - Термометр стресса (Stress Thermometer)
**Концепция:** Термометр, показывающий уровень стресса
- Визуал: Термометр с цветовой индикацией (зеленый-желтый-красный)
- Интерактив: Клик для выбора уровня, перетаскивание
- Анимация: Жидкость в термометре поднимается/опускается
- Шкала: 1-10

### 8. Качество сна - Звездное небо (Star Rating Sleep)
**Концепция:** Выбор качества сна через звезды
- Визуал: Ночное небо со звездами
- Интерактив: Клик на звезды для оценки
- Анимация: Звезды загораются, мерцают
- Шкала: 1-5 звезд

### 9. Активность - Шагомер (Step Counter)
**Концепция:** Интерактивный шагомер с визуализацией шагов
- Визуал: Шагомер с цифрами и индикатором
- Интерактив: Клик для увеличения/уменьшения, ввод с клавиатуры
- Анимация: Цифры переключаются, индикатор движется
- Диапазон: 0-50000 шагов

### 10. Вода - Стакан воды (Water Glass)
**Концепция:** Стакан, который наполняется при выборе количества
- Визуал: Стакан с водой, уровень жидкости
- Интерактив: Клик для добавления стаканов, перетаскивание уровня
- Анимация: Вода наливается, пузырьки
- Единица: Стаканы (250мл) или литры

### 11. Боль - Карта тела (Body Map)
**Концепция:** Интерактивная карта тела для указания локализации боли
- Визуал: Схема тела (вид спереди/сзади)
- Интерактив: Клик на область тела, выбор интенсивности
- Анимация: Область подсвечивается, пульсирует
- Формат: Локализация + интенсивность (1-10)

### 12. Энергия - Батарейка (Energy Battery)
**Концепция:** Батарейка, показывающая уровень энергии
- Визуал: Батарейка с индикатором заряда
- Интерактив: Клик для выбора уровня, перетаскивание
- Анимация: Заряд меняется, цвет индикатора
- Шкала: 0-100%

## UX принципы

1. **Визуальная привлекательность** - каждый элемент должен быть красивым
2. **Интуитивность** - понятно без объяснений
3. **Анимации** - плавные и приятные
4. **Мобильная адаптивность** - работает на всех устройствах
5. **Обратная связь** - визуальная реакция на действия
6. **Доступность** - поддержка клавиатуры и экранных читалок
