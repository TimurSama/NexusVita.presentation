## Как инициализировать профиль Марии:

### Вариант 1: Автоматически (при первом использовании бота)

Когда Мария отправит `/start` боту впервые, профиль создастся автоматически.

### Вариант 2: Вручную через API

После деплоя на Vercel вызовите endpoint:

```bash
POST https://your-project.vercel.app/api/admin/init-maria
```

Или через curl:
```bash
curl -X POST https://your-project.vercel.app/api/admin/init-maria
```

### Вариант 3: Через браузер

Откройте в браузере (после деплоя):
```
https://your-project.vercel.app/api/admin/init-maria
```

И отправьте POST запрос (можно использовать Postman или другой инструмент).

## Проверка создания профиля:

После инициализации проверьте:

1. **Пользователь создан:**
   - Откройте бота в Telegram
   - Отправьте `/start`
   - Должно быть персональное приветствие для Марии

2. **Документ доступен:**
   ```
   GET /api/users/{userId}/documents
   ```
   Должен вернуть документ с описанием проблематики со спиной

3. **План создан:**
   ```
   GET /api/users/{userId}/plans?date=2025-02-15
   ```
   Должен вернуть план на указанную дату

4. **Профиль создан:**
   ```
   GET /api/users/{userId}/profile
   ```
   Должен вернуть профиль с данными Марии

## Что происходит при инициализации:

1. Создается пользователь Мария (если не существует)
2. Создается/обновляется профиль с данными:
   - Дата рождения: 1997-10-23
   - Рост: 172 см
   - Вес: 57 кг
   - Пол: женский
3. Сохраняется медицинский документ с полным описанием проблематики
4. Создается план на весь текущий месяц:
   - Понедельник: 5 упражнений
   - Вторник: 4 упражнения
   - Среда: 5 упражнений
   - Четверг: 4 упражнения
   - Пятница: 5 упражнений
   - Суббота: 5 упражнений
   - Воскресенье: 4 упражнения
5. Настраиваются уведомления Telegram бота:
   - Напоминания в 08:00, 12:00, 18:00, 21:00
   - Все уведомления включены

## Примечания:

- Профиль создается автоматически при первом использовании бота Марией

- Документ сохраняется один раз (не дублируется)
