## Техническая реализация

### Компоненты:
1. `DetailPopup.tsx` - основной компонент попапа
2. `InfoIndicator.tsx` - иконка интерактивности
3. `InteractiveCard.tsx` - карточка с попапом
4. `MetricDetail.tsx` - детали метрики
5. `ModuleDetail.tsx` - детали модуля

### Типы данных:
```typescript
interface PopupContent {
  id: string;
  title: string;
  subtitle?: string;
  description: string;
  sections?: {
    title: string;
    content: string | ReactNode;
  }[];
  visualizations?: ReactNode;
  relatedLinks?: { label: string; path: string }[];
}
```

### Управление состоянием:
- `useDetailPopup()` - хук для управления попапами
- Контекст для глобального состояния
- Локальное состояние для каждого попапа

## UX принципы

1. **Не перегружать страницу** - только ключевая информация
2. **Легкий доступ к деталям** - один клик до попапа
3. **Визуальная индикация** - понятно, что можно кликнуть
4. **Быстрая загрузка** - попапы открываются мгновенно
5. **Плавные анимации** - приятный опыт взаимодействия
6. **Адаптивность** - работает на всех устройствах
7. **Keyboard navigation** - поддержка клавиатуры
8. **Accessibility** - доступность для всех
