# Что еще нужно для полноценного первого релиза (продвижение/инвестиции)

## Продукт и маркетинг
- Лендинг с четким позиционированием и кейсами (до/после).
- Медиакит: брендинг, логотипы, скриншоты, видео‑демо.
- Публичная дорожная карта и обновления.
- PR‑пакет для партнеров (клиники/тренеры/нутрициологи).

## Инвесторский пакет
- Pitch‑deck с цифрами рынка и воронкой.
- Токеномика + финансовая модель (3 года).
- Юр‑структура, прозрачное использование средств.
- План листинга/ликвидности (если нужен).

## Токенсейл
- KYC/AML провайдер подключен и протестирован.
- Страница сейла + whitelist/allowlist.
- Механика скидок и вестинга для ранних.
- Полный аудит смарт‑контрактов (если on‑chain).

## Продуктовые критические блоки
- Полная система ролей и прав доступа.
- Автоматическое начисление достижений/наград.
- Чат со специалистами (реальные сообщения).
- Планирование и напоминания (календарь).

## Комплаенс и доверие
- Политика мед. данных + соглашение пользователя.
- Стандарты безопасности хранения.
- Прозрачный аудит действий (audit log).

## Технические обязательные
- Полный мониторинг и алерты (Sentry/логирование).
- Тесты для критичных сценариев.
- План реагирования на инциденты.
