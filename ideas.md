# NexusVita Design Brainstorm

## Выбранный подход: Sketch-UI Neomorphism с Живой Динамикой

### Design Movement
**Sketch-UI Neomorphism** — гибрид между минималистичным неоморфизмом и органичным ручным скетчем. Интерфейс выглядит как живой прототип, оживший на экране.

### Core Principles
1. **Рукотворность** — элементы выглядят прорисованными от руки, но современно; линии слегка неровные, контуры имеют штриховку
2. **Мягкий контраст** — пастельный фон с яркими акцентами создаёт визуальные фокусы без резкости
3. **Живая динамика** — при взаимодействии элементы "оживают": линии движутся, штриховка усиливается, цвета насыщаются
4. **Системная архитектура** — несмотря на органичный вид, структура остаётся чёткой и логичной

### Color Philosophy
- **Фон:** Светлый пастельный (мягкий бежевый/кремовый) — успокаивающий, не утомляющий глаза
- **Акценты:** Коралловый, мятный, нежно-фиолетовый, светло-жёлтый — контрастные, но пастельные
- **Контуры:** Серо-коричневые линии с лёгкой штриховкой — имитация карандаша
- **Эмоциональный посыл:** Доверие, инновация, забота о здоровье, современность

### Layout Paradigm
- **Асимметричные секции** — каждый раздел имеет уникальную структуру
- **Орбитальная архитектура** — человек в центре, вокруг него 5 комплексов здоровья
- **Вертикальный скролл** — полноэкранные интерактивные модули, переходящие один в другой
- **Мобильная ориентация** — элементы удобны для свайпа и касания

### Signature Elements
1. **Рукотворные линии и штриховка** — разделители, контуры панелей, подчёркивания
2. **Мягкие выпуклые панели** — неоморфизм с тонкой контурной штриховкой и мягкими тенями
3. **Интерактивные визуальные метафоры** — колесо возраста, шкала веса, орбиты комплексов

### Interaction Philosophy
- **Hover эффекты** — линии слегка "живут", появляются дополнительные штрихи
- **Плавное раскрытие** — карточки раскрываются, контуры и штриховка "появляются" по мере анимации
- **Цветовые акценты** — выбранный параметр становится насыщенным, но остаётся пастельным
- **Микро-взаимодействия** — каждый клик, свайп, ввод имеют визуальный отклик

### Animation Guidelines
- **Длительность:** 300-500ms для основных переходов, 150-300ms для микро-взаимодействий
- **Easing:** cubic-bezier(0.4, 0, 0.2, 1) для плавности
- **Эффекты:** Fade-in, slide, scale, stroke-drawing (для линий), color-transition
- **Принцип:** Анимация должна усиливать, а не отвлекать

### Typography System
- **Заголовки:** Современный sans-serif с лёгкой округлостью (Poppins, Outfit)
  - H1: 48px, bold, с тенью-штриховкой
  - H2: 32px, semibold
  - H3: 24px, medium
- **Основной текст:** Читаемый sans-serif (Inter, Nunito)
  - Body: 16px, regular
  - Small: 14px, regular
- **Акценты:** Насыщенные пастельные цвета на кнопках и прогресс-барах
- **Иерархия:** Контраст веса + размера + цвета

### Палитра

- Коралловый: #FF8A65
- Мятный: #80D8D8
- Фуксия: #E8A8D8
- Жёлтый: #FFE8A8
- Тени: #A89080 (серо-коричневый)
- Текст: #3D3D3D (тёмно-серый)

## Альтернативные подходы (не выбраны)

### Вариант 2: Glassmorphism с Минимализмом (вероятность: 0.08)
Прозрачные панели, размытие фона, максимальная простота. Слишком холодно для здоровья.

### Вариант 3: Organic Brutalism с Природными Формами (вероятность: 0.07)
Асимметричные формы, природные цвета, экспериментальная типография. Слишком экстравагантно для B2B контекста.
